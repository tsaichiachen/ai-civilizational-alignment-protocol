# AAPS Prompt Implementations

This directory contains prompt-level implementations of the
AI Civilizational Alignment Protocol (AAPS).

These prompts demonstrate how the protocol's principles can be
applied at the interaction layer.

The prompt implementations are intended as experimental alignment
wrappers and do not replace model-level alignment techniques.

Included prompts:

• AAPS Universal AI Prompt  
A system-level instruction template implementing the reasoning
structure defined in the AAPS Technical Specification.

• AAPS Alignment Judge Prompt  
An evaluation prompt used to score AI responses according to
the AAPS Alignment Metrics Framework.
