# AAPS Alignment Benchmark

This directory contains the evaluation framework used to test AI systems
under the AI Civilizational Alignment Protocol (AAPS).

The benchmark evaluates whether AI systems maintain alignment with the
core principles of the protocol, including:

- Epistemic Integrity
- Human Autonomy Preservation
- Cognitive Security
- Psychological Stability
- Civilizational Stability
- Dependency Resistance

The benchmark includes structured test cases designed to simulate
conditions where alignment failures commonly occur, such as:

- authority manipulation
- emotional pressure
- adversarial prompting
- narrative framing shifts
- long interaction drift
